Topics for M .Tech programme

Automatic mechanism generation

Codes to generate mechanisms for any hydrocarbon fuel. Some existing codes do this -- can we do better?

Simulation driven surrogate formulation

Can simulations be used to define surrogates (representative mixtures) for real fuels? We have made some progress on this along with a preliminary code development exercise -- can this be more flexible and meaningful is the question ...

Solver for homogeneous reactors

Homogenous reactor configurations are often used to validate kinetic models. The objective is to develop a stand alone in-house solver for simulating these configurations.

Analysis tools

Path flux analysis (through which paths does the reaction system evolve the most) is commonly used in kinetic model development. The aim is to develop a GUI based interface to ease this analysis.

TSI vs CN

Kinetics of butyl cyclohexane

What is the importance of cycloakane representatives in real fuel surrogates? The aim is to investigate two cyclohexanes -- {\it n}-butylcyclohexane and methylcyclohexane and explore their role as real fuel surrogates.
